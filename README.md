# What is ravynOS? ![](https://api.cirrus-ci.com/github/ravynsoft/ravynos.svg?branch=main) [![Contributor Covenant](https://img.shields.io/badge/Contributor%20Covenant-2.1-4baaaa.svg)](CODE_OF_CONDUCT.md)

ravynOS is an early-stage open source OS project that aims to provide a similar experience and some compatibility with macOS on x86-64 sytems. It builds on the solid foundations of Darwin/XNU, existing open source packages in the same space, and new code to fill the gaps.

# How to run locally?

1. `npm install`
2. `cp .env.example .env`
3. `npm run dev`
